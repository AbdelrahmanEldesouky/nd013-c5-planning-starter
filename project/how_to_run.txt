Instructions for using websockets between python simulator client (simulatorAPI.py) and the C++ server (cserver)

1. Get the dependancies for cserver and build it

	cd project
	./install-ubuntu.sh
	cd cserver_dir (solution_cubic_spirals_STARTER)
	cmake .
	make
	
2. Running the server and the client
	
	// Log in as student
	su - student
	
	//Go into project folder
	cd nd-c5-planning-starter/project
	
	// Run this script 
	./run_carla.sh
	
3. From a new terminal window, in the same project folder (but with workspace as root) 

	//Go into project folder
	cd nd-c5-planning-starter/project

	//Run this script
	./run_main.sh
